# About Me

Alexander is a data scientist specialising in natural language processing, causality and business analytics. His work focuses on robust ML systems, addressing challenges like distribution shifts using Python (pandas, NumPy, scikit-learn), modern NLP libraries (spaCy, Hugging Face), and experiment tracking / evaluation workflows. 

He works comfortably with relational databases (SQL), data pipelines, and API-driven integrations, including LLM-based tools.

Alexander is passionate about digital privacy and ethical AI development. He loves to contribute to open-source projects, some of which are listed on this page.

Currently based in Amsterdam and open for work.
